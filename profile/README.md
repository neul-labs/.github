# Neul Labs

Building software, ML infrastructure, and developer tooling out in the open. Our repos start life as public R&D projects.

## How We Work

- **Learning in public** – roadmaps, RFCs, and field notes land in the repos, not private docs.
- **Docs-first** – every project ships with a microsite, API references, and upgrade guides.
- **Operational by default** – telemetry hooks, CI templates, and release automation are part of the starter kit.

## Focus Areas

| Discipline | What we publish |
| --- | --- |
| Developer Tooling | Prompt ops, background workers, CLI stacks, and SDK ergonomics. |
| ML Infrastructure | Rust/Python accelerators for LangGraph, LiteLLM, CrewAI, and on-device inference. |
| Operational Excellence | Control planes, observability defaults, workflows for release/telemetry hygiene. |

## Featured Projects

- [**fast-litellm**](https://github.com/neul-labs/fast-litellm) – Rust accelerators for LiteLLM routing, rate limiting, and pooling.
- [**fast-crewai**](https://github.com/neul-labs/fast-crewai) – Drop-in shims that speed up CrewAI memory + storage 2-5x.
- [**fast-langgraph**](https://github.com/neul-labs/fast-langgraph) – Cache + checkpoint primitives for LangGraph, written in Rust.
- [**fastworker**](https://github.com/neul-labs/fastworker) – Brokerless task queues plus a control plane/GUI for Python apps.
